# Excellency Spot - Simple Food Website

Excellency Spot is a simple front-end website created using HTML and CSS. 
This project displays popular snacks and beverages with images, descriptions, 
and an order button. It also includes a basic contact form for users to send their details.

## Features
- Simple and responsive webpage layout
- Food items displayed using cards
- Items included: Tea, Coffee, Samosa, and Pani Puri
- "Order Now" button for each item
- Contact form with Name, Age, and Email fields
- Attractive background and styled design

## Technologies Used
- HTML
- CSS

## Purpose of the Project
This project was created as a beginner practice project to learn basic 
web development concepts such as webpage layout, images, buttons, 
and forms.

## Future Improvements
